# 💼 Prerequisites
To use ReVanced Manager, a certain requirements must be fullfilled

## 📝 Requirements

- Android 8+
- Devices **not** ARMv7

## ⬇️ Installation

- Download and install the latest version of [ReVanced Manager](https://github.com/revanced/revanced-manager/releases/latest)
